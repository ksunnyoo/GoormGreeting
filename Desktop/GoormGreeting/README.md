# 🚀 AI와 함께 만든 첫 페이지 (GoormGreeting)

AI의 도움을 받아 HTML, CSS, JavaScript의 기초를 다지며 제작한 **생애 첫 웹 페이지** 프로젝트입니다.  
세련된 글래스모피즘(Glassmorphism) 스타일의 디자인과 실시간 상호작용 기능을 구현했습니다.

<br>

## ✨ 주요 기능 및 특징
* **실시간 대시보드 시계**: 브라우저 현재 시간을 초 단위로 실시간 반영하여 보여줍니다.
* **로컬스토리지(LocalStorage) 활용**: 입력한 이름과 오늘의 목표가 브라우저에 안전하게 저장되어, 페이지를 새로고침하거나 창을 닫았다가 다시 열어도 데이터가 유지됩니다.
* **사용자 친화적 UX**: 등록 버튼을 클릭하는 것은 물론, 입력 창에서 키보드 `Enter` 키를 눌러도 데이터가 즉시 저장됩니다.
* **저장 피드백 애니메이션**: 이름이나 목표를 등록할 때 배경색이 부드럽게 전환되는 애니메이션 효과를 추가하여 시각적 재미를 더했습니다.

<br>

## 🛠️ 사용 기술 (Tech Stack)
* **HTML5**: 웹 페이지의 기본 구조 설계 및 시맨틱 태그 활용
* **CSS3**: 그라데이션 배경, 글래스모피즘 카드 레이아웃, 반응형 폰트 및 트랜지션 애니메이션 구현
* **JavaScript (Vanilla JS)**: `setInterval`을 이용한 시계 기능, 로컬스토리지 데이터 읽기/쓰기, 키보드 이벤트 제어

<br>

## 💡 이번 프로젝트를 통해 배운 점
1. **개발 환경 이해**: Visual Studio Code 내장 터미널과 Git Bash를 활용해 로컬 환경을 제어하는 방법을 배웠습니다.
2. **Git & GitHub 워크플로우**: `git init`, `git add .`, `git commit`, `git push`로 이어지는 버전 관리와 온라인 저장소 업로드 흐름을 완벽히 이해했습니다.
3. **오류 해결 역량**: 올바른 작업 폴더(Root Directory) 설정의 중요성과 마크다운 문법 내 쌍따옴표 오타 등의 디버깅 과정을 경험했습니다.
